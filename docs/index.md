# Microsoft Agent Framework x Chainlit ハンズオンワークショップ

!!! info "このワークショップについて"
    このハンズオンワークショップは、以下のコンテンツをもとに、Microsoft Agent Framework の2026年3月2日時点での最新バージョン 1.0.0rc2 と Microsoft Foundry を利用したバージョンに更新して作成しています。  
    - [GlobalAICommunity/agentcamp-workshop-agent-framework](https://github.com/GlobalAICommunity/agentcamp-workshop-agent-framework)

このワークショップでは、**Microsoft Agent Framework**、**Chainlit**、および **Microsoft Foundry** を活用して、AI チャットエージェントを段階的に構築する方法を学びます。

各フェーズは前のフェーズの成果を基に進めていく構成になっており、最終的にはツール呼び出しや MCP 連携まで実装します。

---

## 学習目標

- Python と Microsoft Foundry を使った AI チャットエージェントの構築
- Chainlit によるチャット UI の作成
- Microsoft Agent Framework の ChatAgent アーキテクチャの理解
- ツール呼び出し（Tool Calling）の実装
- MCP（Model Context Protocol）によるリモートツール連携

---
